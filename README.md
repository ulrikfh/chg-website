# Castle Horizons site

Files included:
- index.html
- about.html
- contact.html

## Upload to GitHub
1. Create a new repository on GitHub.
2. Upload these three HTML files to the root of the repository.
3. Commit the changes.
4. If you want the site live, enable GitHub Pages:
   - Go to Settings
   - Pages
   - Deploy from branch
   - Select your main branch and root folder
5. Your homepage will load from `index.html` automatically.
